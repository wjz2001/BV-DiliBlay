BV ~~(Bug Video)~~ 是一款 [哔哩哔哩](https://www.bilibili.com) 的第三方 `Android TV`
应用，使用 `Jetpack Compose` 开发，支持 `Android 5.0+`

都是随心乱写的代码，能跑就行。

## 个人优化点

- 你懂的
- 修改默认视频编码为h.265
- 首页改成动态
- ugc名称改成“分区”
- pgc名称改成“影视”
- 删除播放器时左下角信息
- 存在历史记录时自动跳转到断点，按确认键从头播放
- 默认弹幕比例175，透明度70，弹幕区域50%
- 播放完成后自动退出播放器
- up主页和搜索结果卡片增加播放量和弹幕量
- 快进自动播放，快退一次从5s改为10s，和快进保持一致
- 播放完成倒计时5秒再播放下一集
